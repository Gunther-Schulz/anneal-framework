## The prior cut: three fix-shapes — this discriminator governs only the ENFORCEMENT axis
(Cross-session analysis, operator-relayed 2026-06-07.) Before the dose question applies, classify
the failure into one of three shapes — **collapsing them mis-routes the fix**:
1. **Exists but doesn't fire** (loaded-but-inert) — the rule's content is right, it just silently
   doesn't bind. Fix = **enforce** (make it fire). E.g. `skill-craft-antipatterns-loaded-but-inert`,
   `convergence-cycle-mechanical-enforcement`.
2. **Fires, but asks the wrong question** (threat-model blind-spot) — the lens runs and does its job,
   but its content/question is incomplete. Fix = **sharpen the question** — NOT an enforcement or
   dose move. E.g. the co-producer (the Coupled-change pass fired, surfaced the sibling, but asked
   "does a *consumer* parse sub-fields?" not "is there a second *producer*?") — fixed by the
   `enumeration-target-blindness` run.
3. **No mechanism at the scope** — nothing fires at all. Fix = **build a new construct**. E.g. the
   cross-run deferred-removal register (`deferred-removal-cross-run-obligation-ledger`): dead-code
   detection (in-run) + owed-artifact forcing-function (intra-run) both exist, but nothing models a
   *cross-run* obligation.

**This discriminator (below) governs the enforcement axis — shapes #1 (enforce) and #3 (build), plus
the enforcement *residual* of a #2 fix — NOT shape #2's primary fix (sharpen-the-question), which is
content-correctness, not dose.** The seam: a finding is primarily one shape but can leave residue in
another. `enumeration-target-blindness` was a #2 fix that surfaced a #1-shaped residual (INT-4: the new
co-producer falsifier is recognition-gated → can't fire on the unrecognized-sink case) — handled by
*surface-honestly*, not fake-enforce. That is the dose discriminator operating on #2's residual.

## The premise
A "forcing function" = the evidence-bearing-artifact rule applied: following the rule produces
an artifact, not-following produces none, a **separate checker** catches the absence. Its strength
is a gradient (how far faking needs active fabrication vs mere omission). That definition is ALSO
the **dose criterion**: it says when a forcing function is *possible* and when adding one is *fake*.

The two failure modes are asymmetric and anneal already names BOTH — but in four scattered places:
- **Under-dose** (memory/adherence reliance where a structural check was makeable): the cross-run
  gap (`deferred-removal-cross-run-obligation-ledger`); `skill-craft-antipatterns-loaded-but-inert`;
  `enumeration-target-blindness` INT-4 (the recognition-gated co-producer falsifier).
- **Over-dose** (a gate whose artifact is *vacuously satisfiable* → false comfort, worse than none):
  the **form-not-binding** class (`intent-falsification-soundness-sweep`, V-30); the skill-craft
  **additive-reflex** anti-pattern.

## The 4-part warrant (the discriminator, made explicit)
A forcing function is warranted iff:
1. the failure class is **recurring + structural** (n≥2; "try harder to adhere won't fix it") — not a one-off;
2. you can give it a **genuinely evidence-bearing** artifact (an external truth a separate checker re-derives) — NOT a string-is-present;
3. a **separate context** (verify / convergence / a *later run*) can run the check;
4. **if (2) fails → DO NOT force; surface honestly** as `[VERIFIED — surfaced]` (a forcing function you can't make non-vacuous is a surfacer sold as a gate — the form-not-binding failure).

The guardrail against over-use is the SAME rule that motivates use (warrant 2). `enumeration-target-blindness`
is a live exemplar: it forced where makeable (co-producer falsifier, recognized case) AND refused to
over-claim where not (INT-4 → honest residual). The framework's guardrail demonstrably works; the gap is
that the *decision* rests on a good orchestrator noticing, not on an explicit rule.

## The skill-craft relation (why skill-craft is the home)
skill-craft owns "is this rule under- or over-prescribed." The force/surface/prose decision is that
question at the enforcement layer. Today the criterion is distributed across foundations
(evidence-bearing-artifact rule), skill-craft (additive-reflex), the form-not-binding sweep, and
loaded-but-inert — four faces of one discriminator. A single explicit skill-craft discriminator would
unify them (Pareto), so the NEXT INT-4-shaped moment is decided by rule, not by noticing.
