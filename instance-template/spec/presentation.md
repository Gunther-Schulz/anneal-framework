# <Instance> Specification — Presentation

**OPTIONAL — delete this file if your instance does not declare
a presentation slot.**

The framework prescribes no presentation: it mandates no look,
and instances may differ freely. Per
`anneal-framework/instantiation-guide.md` §2, an instance that
wants to steer its look records the intent here. Most instances
leave this slot empty; the framework defines the slot but never
dictates its content.
