# <Instance> Specification — Lens set

The standardized lens set is <domain>'s recurring blind-spots —
the looking an expert does that a novice would not think to.
Each lens fills the lens-entry shape (Name / Question / Scope)
per `anneal-framework/spec/modules.md` §2.1.

Keep the set **closed** — a lens is in it or not — so the
standardized inspection pass can account for every lens, every
cycle. See `anneal-framework/instantiation-guide.md` §4.

If <domain> has no incident history yet, bootstrap the set as
**Path 2** (skill-craft's term): blank-slate hypotheses,
validate-by-use. As real runs surface real incidents, each
incident confirms a lens or adds one.
